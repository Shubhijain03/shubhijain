TABLES

Table consists of rows and columns.But,in HTML we will create table with the help of rows. First we will make one row and then other.

Some of the tags are discussed below:

  1. Table is created by using < table>< /table>

  2. Table row is represented by< tr>< /tr>

  3. Heading of a cell is represented by< th> < /th>

  4. Table data is represented by< td>< /td>

  5. Table caption is defined by< caption>< /caption>

  6. For specifying column properties of each column column tag is used and it is represented by< col>< /col>

  7. For grouping table Header content in the table we generally prefer to use< thead>< /thead>

  8. For grouping table Footer content in the table we generally prefer to use< tfoot>< /tfoot>
