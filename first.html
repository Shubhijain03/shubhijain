We are going to learn the basics of Html. HTML is the language used for website design. This site on HTML will teach you everything you need to learn about html tags, lists, tables, seo etc.

Some of the tags are:

  • For writing Paragraph,we will use paragraph tag< p> < /p>

  • We can use tags for headings also.There are 6 types of headings< h>< /h>

    1. Heading 1

    2. Heading 2

    3. Heading 3

    4. Heading 4

    5. Heading 5
    6. Heading 6
  • Images can be inserted using< img src="here address of the image will be written">
  • Videos can also be inserted using:
    < video width="320" height="240" controls>
    < source src="movie.mp4" type="video/mp4">
  • List can be created. It can be ordered or unordered.

      Types of Unordered List are:

    1. Disc
    2. Square
    3. Circle

      4. Types of Ordered List are:

    4. "1"
    5. "A"
    6. "a"
    7. "I"
    8. "i"
  • For making statements bold we can use< b>< /b>

    This statement is bold.

  • For making statements italic we can use< i>< /i>

    This statement is italicised.

  • For making statements underline we can use< u> < /u>

    This statement is underlined.

  • For using subscript we can use< sub>< /sub>

    Example is given below:
    C + O2 = CO2

  • For using superscript we can use< sup>< /sup>

    Example is given below:
    (a+b)2 = a2 + b2 + 2ab
